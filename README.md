# 逢甲大學 畢業專題

智慧家庭中基於體感偵測器的情緒感測與回饋系統

# 開發環境

Windows

OpenNI 1.5

Eclipse - Java

# 感測器

ASUS Xtion Pro

NeuroSky MindWave Mobile

Accelerometer(PhidgetSpatial)
